About the project:

Weatherly is a modern, lightweight web application that provides **real-time weather information** for any city in the world. 
Built using **Vite** for fast frontend performance, the app fetches data from a public weather API and displays it in a **clean, responsive interface**.
Users can quickly view temperature, humidity, wind speed, and weather conditions, making it practical for everyday use.

It's Features:

-> Real-time weather updates for any city
-> Clean, minimal, and responsive UI
-> Fast load times powered by Vite
-> Easy-to-use input and display layout

Tech Stack:

Frontend: HTML, CSS, JavaScript
Build Tool: Vite
API: Public Weather API
Deployment: Vercel
